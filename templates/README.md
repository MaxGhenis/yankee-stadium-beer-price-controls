# Agent Templates

This directory contains template files used to generate agent-specific documentation (e.g., `CLAUDE.md`, `GEMINI.md`).

**DO NOT DIRECTLY EDIT `CLAUDE.md` OR `GEMINI.md`.**

These files are automatically generated by `scripts/generate_agent_files.py` from `templates/AGENTS.md.tpl`.

To update the agent documentation:
1.  Edit `templates/AGENTS.md.tpl`.
2.  Run `uv run scripts/generate_agent_files.py` to regenerate the live `CLAUDE.md` and `GEMINI.md` files with the latest model values and template changes.
